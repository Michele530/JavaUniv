# JavaFX expérimental

Cette branche contient des tests pour l'affichage 2D avec JavaFX

⚠️ Expérimental : le setup n'est pas très clair et peut varier selon le système (Windows / Linux) car j'ai switché entre Linux et Windows pendant la réalisation du projet
Sans Gradle ou Maven, l'importation des librairies est un peu compliquée, donc JavaFX est directement inclus dans cette branche.
